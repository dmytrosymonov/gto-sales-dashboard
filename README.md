# GTO Sales Dashboard

Веб-дашборд для просмотра аналитики продаж по данным GTO.UA API.

## Данные

- **PAX** — количество пассажиров
- **Total Sales** — общая выручка (в EUR)
- **Total Cost** — общая себестоимость (в EUR)
- **Total Profit** — прибыль (в EUR)

Все суммы конвертируются в евро по курсам из API.

## Режимы

### Date Start
- Только заказы со статусом **confirmed**
- Периоды: Week (7/−7 дней), Month (30/−30), Year (−12/+6 мес.), Quarter (−4/+4 мес.)
- Возможность выбрать свой период

### Date Created
- Все статусы **кроме cancelled**
- Только исторические данные (без будущих дат)
- Выбор периода вручную (От / До)

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на http://localhost:5173

## API Key

API ключ задан в `src/api.js`. Для смены ключа измените константу `API_KEY`.
